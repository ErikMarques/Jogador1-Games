# Jogador1 Games - Sistema de Estoque

Sistema online para gerenciamento de estoque, compras, vendas, custos e lucro da Jogador1 Games.

## Rodar localmente

```bash
npm install
npm run dev
```

## Variáveis de ambiente

Crie um arquivo `.env` com:

```env
VITE_SUPABASE_URL=https://fxntoltmrxkrdpczfibm.supabase.co
VITE_SUPABASE_ANON_KEY=sua_chave_publishable
```

Na Vercel, cadastre essas mesmas variáveis em:

Project Settings → Environment Variables
